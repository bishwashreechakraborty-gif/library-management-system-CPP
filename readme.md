# Library Management System (C++)

A robust, console-based application built using **Object-Oriented Programming (OOP)** principles in C++. This system allows librarians to efficiently manage book records, handle member transactions (issue/return), and search the inventory.

---

## 🚀 Key Features

* **Book Management:** Add and store book details including Title, Author, and unique ID.
* **Search Functionality:** Quickly locate books by their title.
* **Issue & Return System:** Real-time tracking of book availability.
* **Encapsulation:** Data is handled through classes and private members for security and professional structure.

## 🛠️ Technical Stack

* **Language:** C++
* **Paradigms:** Object-Oriented Programming (Classes, Objects, Constructors)
* **Data Structures:** `std::vector` for dynamic storage of records.

## 📋 How to Run

1.  **Clone the repository:**
    ```bash
   git clone https://github.com/bishwashreechakraborty-gif/library-management-system-CPP.git
    ```
2.  **Navigate to the directory:**
    ```bash
    cd library-management-system-CPP
    ```
3.  **Compile the code:**
    Using G++:
    ```bash
    g++ main.cpp -o library_system
    ```
4.  **Execute the program:**
    ```bash
    ./library_system
    ```

---

## 🏗️ Class Structure

* `Book`: Represents the data model (ID, Title, Author, Status).
* `Library`: Handles the business logic (Searching, Issuing, and managing the collection).
